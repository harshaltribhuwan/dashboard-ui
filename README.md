# Dashboard UI

A modern dashboard UI built with React, Material-UI (MUI), Recharts, and Vite.

## Tech Stack
- React 19  
- Material-UI (MUI)  
- Recharts  
- Vite  
- React Router  
- SASS  
- ESLint
